# sol-de-mi-micelio

Una pequeña pieza web nacida para guardar un lugar, una historia y la luz de ciertos encuentros que cambian el rumbo de todo.

Este proyecto reúne una carta visual, una línea de tiempo, música y detalles interactivos pensados para navegarse con calma. No busca explicar demasiado: prefiere insinuar, acompañar y dejar que cada sección diga lo justo.

Está preparado para publicarse en Netlify como una experiencia ligera, íntima y fácil de compartir.

## Qué incluye

- Cartas interactivas con modal de lectura completa.
- Línea de tiempo con momentos importantes.
- Contadores automáticos desde fechas clave.
- Reproductor musical integrado.
- Mini juego al final de la página.

## Personalización rápida

Los textos, fechas y música se editan directamente desde [index.html](index.html).
Las imágenes y recursos visuales están en [components/media](components/media).

## Nota

Si llegaste hasta aquí, probablemente ya entiendes que esto no es solo una página.
